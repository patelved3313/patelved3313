# One-time setup

1. Go to GitHub and create a new **public** repository named exactly:

   `patelved3313`

2. Do not add a template, license, or `.gitignore`.

3. Upload:
   - `README.md`
   - the complete `assets` folder

4. Commit the files to `main`.

GitHub automatically recognizes a repository whose name matches the username and displays its README on the profile Overview page.
